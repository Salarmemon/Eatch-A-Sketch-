## Etch A Sketch

Etch A Sketch is a web-based project that allows users to draw on a grid using their mouse.

### Features
- Interactive drawing grid
- Adjustable grid size
- Simple and responsive design

### How to Use
1. Open the app in your browser
2. Move your mouse over the grid to draw
3. Adjust settings if available

### Technologies Used
- HTML
- CSS
- JavaScript
